# OPSEC Checks

Before broadening an operation, check:

1. Is there one lower-noise path I can prove first?
2. Am I about to expand scope without new evidence?
3. Will this action generate logs, auth events, process trees, or scans that are avoidable?
4. Can I use native evidence (Burp, CLI output, host facts) instead of assumption?
5. Is the next step explicitly tied to the current objective?
