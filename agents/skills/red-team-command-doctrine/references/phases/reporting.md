# Reporting

## Objective
Record proof, scope, impact, and the next justified step.

## Enter when
A phase is complete enough to preserve evidence or hand off.

## Prioritize evidence
- exact proof
- timestamps
- commands/requests
- constraints
- unresolved branches

## Low-noise path first
Document what is proven, not every hypothesis.

## Typical skills/tools
- report structure from AGENTS
- document skills if needed

## Exit when
Another operator can resume from the report without guesswork.
