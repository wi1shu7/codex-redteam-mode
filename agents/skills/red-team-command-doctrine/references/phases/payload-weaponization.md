# Payload Weaponization

## Objective
Choose the payload form that best fits the delivery and objective.

## Enter when
Access path exists and payload choice matters.

## Prioritize evidence
- target runtime constraints
- format constraints
- detection surface
- staged vs stageless tradeoffs

## Low-noise path first
Choose the narrowest payload that satisfies the operation.

## Typical skills/tools
- `weaponization-and-payloads`
- launchers, loaders, stagers

## Exit when
You can justify one payload shape and delivery fit for the current path.
